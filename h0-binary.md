# h0 - Binary

My first task was to write a short program, compile it to binary or bytecode and run it.
To do this task, I executed a simple Java code in the file Geeks.java in windows notepad: 

public class Geeks{
    
    public static void main(String[] args){
        
        System.out.println("Hello, Geeks");
    }
}

Next, in the command terminal, I compiled the file Geeks.java using javac Geeks.java which in turn generated a file called Geeks.class.
