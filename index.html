ISMS Scope

Scope

This ISMS scope covers the following components within my home network and study environment. The ISMS covers the normal operation, access control, data storage, synchronization, and account security of the defined environment.

In-scope assets

  • Infrastructure: DNA router, Wi-Fi network
  • Devices:
    • HP Windows laptop (primary device for coursework, assignments, and technical exercises)
    • iPhone 13 (used for cloud access and MFA)
    • Kindle (used for reading coursework sometimes)
  • Information assets: Course materials, personal notes, repositories, and all associated accounts, credentials, and MFA

Exclusions

Out of scope

  • Nintendo Switch (used mainly for gaming)
  • Housemates’ devices, as they are outside my ownership and control
  • Infrastructure on the internet side of the ISP router, which cannot be controlled by me

Key Interfaces

  • Cloud services: GitHub, iCloud, OneDrive
  • Remote access: VPN, SSH
  • External boundary: Router connecting the home network to the internet
  • Suppliers: ISP (DNA), device vendors, and cloud service providers
